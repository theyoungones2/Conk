# Link

Once you have successfully run:
- makegst
- make all
- make asm

You can then finally link it all into the exeuctable runtime.

```
Work:Dev/conk/Gonk> link
Slink - Version 6.50
Copyright (c) 1988-1993 SAS Institute, Inc.  All Rights Reserved.


SLINK Complete - Maximum code size = 289624 ($00046b58) bytes

Final output file size = 309192 ($0004b7c8) bytes
```

You can then run your executable `Gonk` or similar.

## Troubleshooting

Error:
```
Work:Dev/conk/Gonk> link
Slink - Version 6.50
Copyright (c) 1988-1993 SAS Institute, Inc.  All Rights Reserved.

ERROR: Multiply defined symbol '_DOSBase'.
       First defined in file 'LIB:c.o'.
        Type = RELOCATABLE   Value = 4
       Redefined in file 'Code/GlobalVars.o'.
        Type = RELOCATABLE   Value = 8

SLINK Complete - Maximum code size = 288428 ($000466ac) bytes

Final output file size = 307964 ($0004b2fc) bytes
SLINK failed returncode 101
```
Remove the definition from [GlobalVars.c](../../Gonk/Source/GlobalVars.c)
```
// struct DosLibrary * __near DOSBase;`
```
This might come back and bite me later, but it seems to work.
