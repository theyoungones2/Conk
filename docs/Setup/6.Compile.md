# Compile

Initial Instructions focus on Gonk

## Gonk
Open NewShell
```
cd CK:Gonk

# Make Script files executable. Avoids having to use "execute makegst"
protect makegst +s add
protect make +s add
protect link +s add

# Make drawer for compiled code *.o
makedir code
```

### Make the Global Symbol Table

### Compile C Code
There's 3 `make` parameters.
- `make all` - Recompile only modified files.
- `make recomp` - Recompile everything
- `make asm` - Assemble assembly language, covered later
```
Work:Dev/conk/Gonk> make all

SAS/C Amiga Compiler 6.50
Copyright (c) 1988-1993 SAS Institute Inc.
SAS/C Amiga Compiler 6.50
Copyright (c) 1988-1993 SAS Institute Inc.
```
